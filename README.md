# Machine Learning Approach for miRNA target site prediction
## Summary
This is a project for my Master Thesis, where the goal is to train a machine/deep learning model that predicts whether a microRNA binds to a target mRNA.
The dataset used was obtained from ***et.al*** and is based on results from an AGO2-eCLIP experiment from ***Manakov et.al. 2022***. Negative samples were generated for this dataset in order to mitigate class imbalance via a clustering based sampling method implemented by ***et.al*** and the dataset was further corrected for false negative samples based on data from Tarbase that contains experimentally validated miRNA-mRNA interactions.
